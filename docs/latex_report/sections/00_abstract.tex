\section*{Abstract}

This project presents an end-to-end big data analytics pipeline for predicting arrest likelihood from over 8.5 million crime records spanning 2001--2026 from the City of Chicago. Following the CRISP-DM methodology, we designed and implemented a full data architecture encompassing data ingestion via Socrata API and Sqoop, optimized storage in Hive with Parquet+Snappy format, feature engineering using custom PySpark transformers, machine learning model training with hyperparameter tuning on a Hadoop YARN cluster, and interactive visualization through Apache Superset. Two classification models were evaluated: Random Forest and Gradient Boosted Trees (GBT), with GBT achieving the best performance (AUC-PR=0.8176, F1=0.8710). The final deliverables include a reproducible GitHub repository, a comprehensive report, and a multi-tab Superset dashboard presenting data description, exploratory analysis, and ML modeling results.
